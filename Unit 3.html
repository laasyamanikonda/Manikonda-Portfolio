Unit 3

What Is This?

This project aimed to create a game that taught users about trash waste and how it affects the planet, as well as general sustainability. Users drag the boat to pick up trash; once they have picked up all the trash, the game ends and the user is taken to a win screen.

What Did I Learn? What Does It Show?

Overall, I had fun doing this project. I enjoyed creating the background and the trash, as it was fun to play with various colors, shapes, shape properties, etc. and generally gave me a sense of increased creativity for the projects thereafter. In addition to new coding skills, I learned collaborative skills, as well as time management and prioritization.

How was it created

Although we had not yet covered “conditionals” in this unit, my partner and I used nested conditionals and onMouseDrag to create this game, as well as numerous functions for the trash in the ocean. Creating these functions was difficult, but ended up being more efficient in the long run; I simply called the functions a few times, and the 'ocean' was filled with trash.
